# Comércio Internacional e Crescimento Econômico

Este repositório contém os dados e scripts utilizados na dissertação
"Comércio Internacional e Crescimento Econômico: uma análise em painel
para países dependentes e não dependentes de commodities (2000–2022)".

## Estrutura do repositório
- data/: bases de dados brutas e tratadas
- scripts/: scripts em R para replicação dos resultados
- outputs/: tabelas e figuras geradas

## Como replicar os resultados
1. Abrir o RStudio
2. Executar o script scripts/00_setup.R
3. Executar os scripts na ordem numérica

## Dependências
R (>= 4.2.0)  
Pacotes: tidyverse, plm, ggplot2, lmtest, sandwich
