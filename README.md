Este diretório contém os dados e scripts utilizados na dissertação:

"Comércio Internacional e Crescimento Econômico:
uma análise em painel para países dependentes e não dependentes de commodities (2000–2022)"

Conteúdo:
- Dados.xlsx: base de dados utilizada no estudo.
- script_dissertacao.R: estimações econométricas (painel).
- script_graficos.R: geração de gráficos descritivos.

Reprodutibilidade:
1. Abrir o RStudio
2. Definir o diretório de trabalho para esta pasta
3. Executar os scripts na ordem:
   (i) script_dissertacao.R
   (ii) script_graficos.R

Versão do R utilizada: R 4.2 ou superior
Pacotes necessários: tidyverse, plm, lmtest, sandwich, ggplot2
